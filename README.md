# 🧹 Data Cleaning Project — Customer Personality Analysis

## Overview
Cleaned a real-world marketing dataset of 2240 customers 
using Python and Pandas.

## What Was Done
- Handled 24 missing Income values using median imputation
- Removed extreme outliers (Income: 666,666)
- Standardized inconsistent Marital Status categories 
  (YOLO, Absurd, Alone → Single)
- Converted date columns to proper DateTime format
- Dropped constant/useless columns
- Engineered new features: Age, Customer_Days

## Key Insights
- Visualized Income distribution, Age, Marital Status, 
  and Spending patterns
- Dataset reduced from 29 → 27 columns with higher data quality

## Tools Used
- Python, Pandas, NumPy, Matplotlib, Seaborn
